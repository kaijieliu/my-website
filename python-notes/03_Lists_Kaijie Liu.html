Lists

A list is a collection of items in a specific order. You can store anything in a list — letters, numbers, or even the names of schools. List items don’t have to be related; Python simply keeps them together so you can access or modify them later. In Python,

  • Square brackets [] indicate a list

  • Items are separated by commas.

  • Because lists usually contain multiple values, it’s good practice to give them plural names, such as names, IDs, schools.

Here’s a simple example of a list of schools.

schools = ['ABC Elementary School', 'DEF Middle SChool', 'GHI High School']

print(schools)
['ABC Elementary School', 'DEF Middle SChool', 'GHI High School']

Assess an element in a list

Each item in a list has a position, called an index. Because a list is ordered, we can retrieve (or access) any element by writing the list’s name followed by the element’s index inside square brackets.

In Python, indexing starts at 0, not 1.
Using the schools list from above, the index-to-element relationship looks like this:

Index Element
0 ‘ABC Elementary School’
1 ‘DEF Middle School’
2 ‘GHI High School’

The index of a list starts from 0 for the first element. From the above example, schools, we see how each index correspond to the elements and how the print(schools[3]) command spit out error (i.e., there are only three elements correspond to index/position 0, 1, 2).

print(schools[0])
ABC Elementary School
print(schools[1])
DEF Middle SChool
print(schools[2])
GHI High School

If you try to access an index that does not exist, Python gives an error:

print(schools[3])
IndexError: list index out of range

Assess the first and last element in a list

By now, it should be clear that you can access the first element in a list by using the index 0.

Python also provides a convenient way to access elements from the end of a list. If you use a negative index, Python counts backward:

  • -1 → last element

  • -2 → second-to-last element

  • -3 → third-to-last element and so on

print(schools[0])
ABC Elementary School
print(schools[-1])
GHI High School
print(schools[-2])
DEF Middle SChool
print(schools[-3])
ABC Elementary School

Use string methods on list elements

We can apply string methods on a string element in the list. For example,

print(schools[0].upper())
ABC ELEMENTARY SCHOOL
print(schools[0].lower())
abc elementary school
print(schools[0].title())
Abc Elementary School

Use Individual Values from a List

You can use individual values from a list just as you would use any other variable in Python. This includes inserting them into f‑strings to build readable messages. For example, using the schools list:

message = f"{schools[0].title()} is a great school."

print(message)
Abc Elementary School is a great school.

Say, if we also have the numbers of students stored in the same order in a list called counts.

counts = [356, 1200, 2300]

message = f"{schools[0].title()} is a great school. {counts[0]} students learn and grow happily there."

print(message)
Abc Elementary School is a great school. 356 students learn and grow happily there.

A numerical example

Accessing a list element and doing math with it does NOT change the list. To actually modify the list, you must assign the new value:

counts = [10, 20, 30]

# This calculates 10 + 5 but does NOT store it
counts[0] + 5
15
print("After counts[0] + 5:", counts)
After counts[0] + 5: [10, 20, 30]
# This actually updates the list
counts[0] = counts[0] + 5
print("After assignment:", counts)
After assignment: [15, 20, 30]

Making Changes to a list

Most lists you create will be dynamic, meaning you’ll build a list and then add, change, or remove elements as your program runs. This makes lists one of the most flexible and commonly used data structures in Python.

Modify Elements in a List

The syntax for modifying an element is similar to the syntax for accessing an element. To change an item, write the list’s name followed by the index of the element you want to update, and then assign a new value.

Using the schools list as an example, let’s explore how modifications work.

schools[0] = "Tree House Elementary School" 

schools
['Tree House Elementary School', 'DEF Middle SChool', 'GHI High School']
schools[1] = "Sun Shine Middle School" 

schools
['Tree House Elementary School', 'Sun Shine Middle School', 'GHI High School']
schools[2] = "Golden Star High School" 

schools
['Tree House Elementary School', 'Sun Shine Middle School', 'Golden Star High School']

Add Elements to a List

In general, there are two ways to add elements to a list.

  • Appending elements to the end of a list

  • Inserting elements into a list

append()

When you append an item, Python places it at the end of the list.

schools.append("Blue River Technical School") 

schools
['Tree House Elementary School', 'Sun Shine Middle School', 'Golden Star High School', 'Blue River Technical School']
schools.append("Blue River Technical School", "Random School") # This won't work for adding them both together
TypeError: list.append() takes exactly one argument (2 given)
schools
['Tree House Elementary School', 'Sun Shine Middle School', 'Golden Star High School', 'Blue River Technical School']
# one way to do this
new_schools = ["Blue River Technical School", "Random School"]

schools = schools + new_schools

schools
['Tree House Elementary School', 'Sun Shine Middle School', 'Golden Star High School', 'Blue River Technical School', 'Blue River Technical School', 'Random School']
# another way
schools.extend(["Blue River Technical School", "Random School"])
schools
['Tree House Elementary School', 'Sun Shine Middle School', 'Golden Star High School', 'Blue River Technical School', 'Blue River Technical School', 'Random School', 'Blue River Technical School', 'Random School']

If we run this one more time, a duplicated “Blue River Technical School” will be added to the end.

schools.append("Blue River Technical School") 

schools
['Tree House Elementary School', 'Sun Shine Middle School', 'Golden Star High School', 'Blue River Technical School', 'Blue River Technical School', 'Random School', 'Blue River Technical School', 'Random School', 'Blue River Technical School']

insert()

When we want the added element to be at a specific position, we can use insert(). We do this by specifying the index of the new element and the value of the new item. Python will automatically shift the remaining items to make room for the new one.

schools.insert(0, "Beaver Elementary School") 

schools
['Beaver Elementary School', 'Tree House Elementary School', 'Sun Shine Middle School', 'Golden Star High School', 'Blue River Technical School', 'Blue River Technical School', 'Random School', 'Blue River Technical School', 'Random School', 'Blue River Technical School']
schools.insert(2, "Moonlight Middle School") 

schools
['Beaver Elementary School', 'Tree House Elementary School', 'Moonlight Middle School', 'Sun Shine Middle School', 'Golden Star High School', 'Blue River Technical School', 'Blue River Technical School', 'Random School', 'Blue River Technical School', 'Random School', 'Blue River Technical School']

Remove Elements from a List

You can remove an item by its position in the list or by its value. Python provides different tools depending on what you need to do. Each method affects the list in a slightly different way.

Remove an item by its position with del

For example, if we want to remove the last duplicated school, we can do the following.

del(schools[-1])
print(schools)
['Beaver Elementary School', 'Tree House Elementary School', 'Moonlight Middle School', 'Sun Shine Middle School', 'Golden Star High School', 'Blue River Technical School', 'Blue River Technical School', 'Random School', 'Blue River Technical School', 'Random School']

Or, if we want to delete the first element.

del(schools[0])
print(schools)
['Tree House Elementary School', 'Moonlight Middle School', 'Sun Shine Middle School', 'Golden Star High School', 'Blue River Technical School', 'Blue River Technical School', 'Random School', 'Blue River Technical School', 'Random School']

Remove an item and get its value with pop()

Sometimes you’ll want to use the value of an item after you remove it from a list. The pop() method is perfect for this situation. It removes the last item in a list by default, and it returns that item so you can store it or use it in your program.

The term pop comes from imagining a list as a stack of items. When you “pop” an item, you remove the one on top. In this analogy, the top of the stack corresponds to the end of the list.

In the example below, we start by defining and printing the list schools. Then we pop a value from the list, and assign that value to the variable popped_schools. Then we print the list, popped_schools, to prove that we still have access to the value that was removed.

schools = ['Beaver Elementary School', 'Tree House Elementary School', 'Moonlight Middle School', 'Sun Shine Middle School', 'Golden Star High School', 'Blue River Technical School']

print(schools)
['Beaver Elementary School', 'Tree House Elementary School', 'Moonlight Middle School', 'Sun Shine Middle School', 'Golden Star High School', 'Blue River Technical School']
popped_schools = schools.pop()

print(popped_schools)
Blue River Technical School

This can be very useful when we want to know the last element in a list.

Remove an item by its value with remove()

Sometimes you won’t know the position of the item you want to remove from a list. If you only know the value, you can use the remove() method. Python will search the list and remove the first occurrence of the matching value.

If the value appears more than once, remove() deletes only the first match. If the value is not in the list, Python raises an error, so it’s often useful to check before removing. See the examples below.

schools = ['Beaver Elementary School', 'Tree House Elementary School', 'Blue River Technical School', 'Beaver Elementary School']

print(schools)
['Beaver Elementary School', 'Tree House Elementary School', 'Blue River Technical School', 'Beaver Elementary School']
schools.remove('Beaver Elementary School')

print(schools)
['Tree House Elementary School', 'Blue River Technical School', 'Beaver Elementary School']
schools.remove('Blue River Elementary School')
ValueError: list.remove(x): x not in list

Where does the error come from? We have ‘Blue River Technical School’ rather than ‘Blue River Elementary School’ in the list.

We can also use the remove() method to work with a value that’s being removed from a list. See how the variable, duplicate, is created and used in the f-string below.

schools = ['Beaver Elementary School', 'Tree House Elementary School', 'Beaver Elementary School']

print(schools)
['Beaver Elementary School', 'Tree House Elementary School', 'Beaver Elementary School']
duplicate = 'Beaver Elementary School'

schools.remove(duplicate)

print(schools)
['Tree House Elementary School', 'Beaver Elementary School']
print(f"{duplicate} is listed twice and so, we are cleaning the duplicate.")
Beaver Elementary School is listed twice and so, we are cleaning the duplicate.

Organizing a List

Often, your lists will be created in an unpredictable order. Sometimes, it might be OK to preserve the original order of your list, while other times, we might want to change that order entirely. Python provides several useful tools to organize lists, depending on our needs — whether we want a temporary, sorted view of the list or a permanent reordering of the items.

Sort a List Permanently with the sort() Method

The sort() method makes it relatively easy to sort a list. It sorts the list in an alphabetical order and permanently changes the order of a list. See how it works below.

schools = ['Beaver Elementary School', 'Tree House Elementary School', 'Moonlight Middle School', 'Sun Shine Middle School', 'Golden Star High School', 'Blue River Technical School']

schools.sort()

print(schools)
['Beaver Elementary School', 'Blue River Technical School', 'Golden Star High School', 'Moonlight Middle School', 'Sun Shine Middle School', 'Tree House Elementary School']

A numerical example:

scores = [100, 97, 94, 88]
print(scores)
[100, 97, 94, 88]
scores.sort()
print(scores)
[88, 94, 97, 100]

We can also sort this list in reverse-alphabetical order by passing the argument reverse=True to the sort() method.

schools = ['Beaver Elementary School', 'Tree House Elementary School', 'Moonlight Middle School', 'Sun Shine Middle School', 'Golden Star High School', 'Blue River Technical School']

schools.sort(reverse=True)

print(schools)
['Tree House Elementary School', 'Sun Shine Middle School', 'Moonlight Middle School', 'Golden Star High School', 'Blue River Technical School', 'Beaver Elementary School']

A numerical example:

scores = [100, 97, 94, 88]
print(sorted(scores))
[88, 94, 97, 100]
print(scores) # scores are still in the original order
[100, 97, 94, 88]

Sort a List Temporarily with the sorted() Function

To maintain the original order of a list but present it in a sorted order, you can use the sorted() function. The sorted() function lets us display our list in a particular order, but doesn’t affect the actual order of the list.

schools = ['Beaver Elementary School', 'Tree House Elementary School', 'Moonlight Middle School', 'Sun Shine Middle School', 'Golden Star High School', 'Blue River Technical School']

print("Here is the original list: ")
Here is the original list: 
print(schools)
['Beaver Elementary School', 'Tree House Elementary School', 'Moonlight Middle School', 'Sun Shine Middle School', 'Golden Star High School', 'Blue River Technical School']
print("\nHere is the sorted list: ")

Here is the sorted list: 
print(sorted(schools))
['Beaver Elementary School', 'Blue River Technical School', 'Golden Star High School', 'Moonlight Middle School', 'Sun Shine Middle School', 'Tree House Elementary School']
print("The original list is still here: ")
The original list is still here: 
print(schools)
['Beaver Elementary School', 'Tree House Elementary School', 'Moonlight Middle School', 'Sun Shine Middle School', 'Golden Star High School', 'Blue River Technical School']

The sorted() function can also accept a reverse=True argument.

schools = ['Beaver Elementary School', 'Tree House Elementary School', 'Moonlight Middle School', 'Sun Shine Middle School', 'Golden Star High School', 'Blue River Technical School']

print("Here is the original list: ")
Here is the original list: 
print(schools)
['Beaver Elementary School', 'Tree House Elementary School', 'Moonlight Middle School', 'Sun Shine Middle School', 'Golden Star High School', 'Blue River Technical School']
print("\nHere is the list in reverse-alphabetical order: ")

Here is the list in reverse-alphabetical order: 
print(sorted(schools, reverse = True))
['Tree House Elementary School', 'Sun Shine Middle School', 'Moonlight Middle School', 'Golden Star High School', 'Blue River Technical School', 'Beaver Elementary School']

Print a List in Reverse Order

To reverse the original order of a list, we can use the reverse() method. However, reverse() doesn’t sort backward alphabetically; it simply reverses the order of the list:

schools = ['Beaver Elementary School', 'Tree House Elementary School', 'Moonlight Middle School', 'Sun Shine Middle School', 'Golden Star High School']

print("Here is the original list: ")
Here is the original list: 
print(schools)
['Beaver Elementary School', 'Tree House Elementary School', 'Moonlight Middle School', 'Sun Shine Middle School', 'Golden Star High School']
schools.reverse()

print("\nHere is the reversed list: ")

Here is the reversed list: 
print(schools)
['Golden Star High School', 'Sun Shine Middle School', 'Moonlight Middle School', 'Tree House Elementary School', 'Beaver Elementary School']

Find the Length of a List

The len() function is useful when we want to identify the length of a list.

schools = ['Beaver Elementary School', 'Tree House Elementary School', 'Moonlight Middle School', 'Sun Shine Middle School', 'Golden Star High School']

len(schools)
5

A Numerical example

# sorting
scores = [78, 92, 85, 88, 69]
print(scores)
[78, 92, 85, 88, 69]
print(sorted(scores))
[69, 78, 85, 88, 92]
print(scores)
[78, 92, 85, 88, 69]
scores.sort()
print(scores)
[69, 78, 85, 88, 92]
# highest
print(scores[-1])
92
# lowest
print(scores[0])
69
# reverse
scores = [78, 92, 85, 88, 69]
print(scores)
[78, 92, 85, 88, 69]
print(sorted(scores,reverse = True))
[92, 88, 85, 78, 69]
print(scores)
[78, 92, 85, 88, 69]
scores.sort(reverse = True)
print(scores)
[92, 88, 85, 78, 69]

Index Errors When Working with Lists

A common index error happens when we are trying to ask for the last item in a list but input a wrong index. Say, we are working with the above list, schools, for the first time, and we saw its length being 5, so we ask for the last item by typing the syntax below.

schools[5]
IndexError: list index out of range

This results in an index error, since for a list of length of 5, the last valid index is 4. Python’s indexing starts at 0, so a list with five items is indexed as 0, 1, 2, 3, 4. Trying to access index 5 is outside the valid range and triggers the error. One way to avoid this type of error is to use the index -1 instead, which always returns the last item in a list.

schools[-1]
'Golden Star High School'

The only time this approach will cause an error is when you request the last item from an empty list:

schools_emp = []
schools_emp[-1]
IndexError: list index out of range

Note:

If an index error occurs and you’re not sure why, a good troubleshooting strategy is to print your list or print the length of the list. Sometimes the list may not look the way you expect—especially if it has been built or modified dynamically by your program. Seeing the actual items, or knowing exactly how many elements are in the list, can help you identify and fix the logical error.

Tuples

Lists work well for storing collections of items that can change throughout the life of a program. The ability to modify lists is especially important when you’re working with data that must be updated frequently. However, sometimes you’ll want to create a sequence of items that should not change. Python refers to values that cannot change as immutable, and an immutable list is called a tuple.

A tuple looks similar to a list, but instead of using square brackets [], it uses parentheses ().

scores = (100, 97, 94, 88, 86, 79)
print(scores)
(100, 97, 94, 88, 86, 79)
# this won't work
scores[0] = 99
TypeError: 'tuple' object does not support item assignment
