What is pandas?

pandas is a powerful, open‑source Python library designed specifically for data cleaning, manipulation, and analysis. It provides intuitive, high‑performance data structures that make it easier to work with tabular or labeled data. pandas is built on top of NumPy, giving it the ability to handle vectorized (column-wise) operations without loops, which is essential when working with thousands or millions of cases.

Introduction to pandas Data Structures

There are two basic data structures:

  • Series

  • DataFrame

Series

A Series is a one-dimensional array-like object containing an array of data (of any NumPy data type) and an associated array of data labels, called the index. The simplest Series is formed from only an array of data like the followings.

v = pd.Series([3, 6, 4, 5])

v
0    3
1    6
2    4
3    5
dtype: int64

Notice that output of v above shows the index on the left and the values on the right. Since we did not specify an index for the data, a default one consisting of the integers 0 through N - 1 (where N is the length of the data) is created. We can get the array representation and index object of the Series via its values and index attributes, respectively:

v.values
array([3, 6, 4, 5])
v.index
RangeIndex(start=0, stop=4, step=1)

We can create a Series with an index identifying each data point, which is more desirable.

series = pd.Series([-1, 2, 3, 5], index = ['a', 'b', 'c', 'd'])

series
a   -1
b    2
c    3
d    5
dtype: int64
series.values
array([-1,  2,  3,  5])
series.index
Index(['a', 'b', 'c', 'd'], dtype='str')

We can use values in the index when selecting single values or a set of values

series[['b']]
b    2
dtype: int64
series[['c', 'd']]
c    3
d    5
dtype: int64

NumPy array operations, such as filtering with a boolean array, scalar multiplication, or applying math functions, will preserve the index-value link.

series[series > 0]
b    2
c    3
d    5
dtype: int64
series*3
a    -3
b     6
c     9
d    15
dtype: int64

Another way to think about a Series is as a fixed-length, ordered dictionary, as it is a mapping of index values to data values. It can be substituted into many functions that expect a dictionary.

'c' in series
True
'e' in series
False

If we have data contained in a Python dictionary, we can create a Series from it by passing that dictionary to pd.Series().

# Dictionary of assessment scores
scores_dict = {
    "StudentA": 92,
    "StudentB": 85,
    "StudentC": 78
}

# Create a Series from the dictionary
scores_series = pd.Series(scores_dict)

print(scores_series)
StudentA    92
StudentB    85
StudentC    78
dtype: int64

DataFrame

A DataFrame is a two‑dimensional, table‑like data structure commonly used in Python libraries such as pandas. We can think of it as a spreadsheet, but with much more flexibility.

  • Tabular structure: Data is organized into rows and columns.

  • Columns can have different data types: Each column is a Series, and each Series can store values of its own type—numbers, strings, booleans, dates, etc.

  • Indexed: A DataFrame has both:

    • a row index (labels for rows)
    • a column index (labels for columns)

  • Dictionary‑like: You can think of a DataFrame as a dictionary of Series objects, all sharing the same row index.

  • Efficient storage under the hood: Instead of storing each column separately (like a list of arrays), pandas stores data in one or more 2‑dimensional blocks, allowing for fast operations.

One of the most common ways to construct a DataFrame is from a dictionary of equal-length lists or NumPy arrays. Let’s elaborate that with an example below.

Basic construction from a dictionary

Suppose we are storing results from an assessment. We can pass the dictionary containing the data to pd.DataFrame().

import pandas as pd

data = {
    'student': ['Kitty', 'Kitty', 'Bob', 'Bob', 'Jerry'],
    'exam':    ['Midterm', 'Final', 'Midterm', 'Final', 'Midterm'],
    'score':   [88, 92, 75, 81, 90]
}

frame = pd.DataFrame(data)
frame
student exam score
0 Kitty Midterm 88
1 Kitty Final 92
2 Bob Midterm 75
3 Bob Final 81
4 Jerry Midterm 90

Notice that

  • The index (0, 1, 2, …) is assigned automatically

  • The columns appear in the order given in the dictionary

Specifying column order

Say, we now want to change the order of the columns. We can pass an explicit sequence of columns. The DataFrame’s columns will be exactly what we specify.

frame2 = pd.DataFrame(data, columns=['exam', 'student', 'score'])

frame2
exam student score
0 Midterm Kitty 88
1 Final Kitty 92
2 Midterm Bob 75
3 Final Bob 81
4 Midterm Jerry 90

Adding columns that don’t exist yet

Suppose we now want to specify a column that doesn’t exist in the data above. We can specify that column in the pd.DataFrame() function. Below, see how we do it and how missing values show up in the results. Additionally, see how we can customize row labels (i.e., index), for example, by adding “record”.

frame3 = pd.DataFrame(
  data, 
  columns = ['exam', 'student', 'score', 'passed'],
  index = ['record1', 'record2', 'record3', 'record4', 'record5'])

frame3
exam student score passed
record1 Midterm Kitty 88 NaN
record2 Final Kitty 92 NaN
record3 Midterm Bob 75 NaN
record4 Final Bob 81 NaN
record5 Midterm Jerry 90 NaN

Note: - The new column passed didn’t exist in data, so it’s filled with NaN - records have been added as indices

Assessing a row

We can assess a row by its index label using .loc.

frame3.loc['record3']
exam       Midterm
student        Bob
score           75
passed         NaN
Name: record3, dtype: object

Modifying a column

Let’s say, a student “passed” if score >= 85. We can define the passed column as follows.

frame3["passed"] = frame3["score"] >= 85

frame3
exam student score passed
record1 Midterm Kitty 88 True
record2 Final Kitty 92 True
record3 Midterm Bob 75 False
record4 Final Bob 81 False
record5 Midterm Jerry 90 True

Or, if we want to change the score column, we can do this:

frame3["score"] = [93, 95, 70, 80, 88]

frame3
exam student score passed
record1 Midterm Kitty 93 True
record2 Final Kitty 95 True
record3 Midterm Bob 70 False
record4 Final Bob 80 False
record5 Midterm Jerry 88 True

If the changes are made only to specific cases, we can assign a Series firstly and pass that into the column.

updated_score = pd.Series([70, 70, 70, 70], index = ["record1", "record2", "record4", "record5"])
frame3["score"] = updated_score

frame3
exam student score passed
record1 Midterm Kitty 70.0 True
record2 Final Kitty 70.0 True
record3 Midterm Bob NaN False
record4 Final Bob 70.0 False
record5 Midterm Jerry 70.0 True

Notice above that the score of “record3” was not specified in the updated_score, so it returns to an NaN. This means, when we assign a Series to the column, the new Series is confirmed exactly to the DataFrame’s index, where missing values will be inserted for those not confirmed.

Attention

Any in-place modification made to a column will be reflected in the DataFrame, since the column is a view on the underlying data, not an independent copy. To make sure we are working safely, it’s suggested to copy the column/series explicitly.

# recreate frame3
data = {
    'student': ['Kitty', 'Kitty', 'Bob', 'Bob', 'Jerry'],
    'exam':    ['Midterm', 'Final', 'Midterm', 'Final', 'Midterm'],
    'score':   [88, 92, 75, 81, 90]
}

frame3 = pd.DataFrame(
  data, 
  index = ['record1', 'record2', 'record3', 'record4', 'record5'])

frame3
student exam score
record1 Kitty Midterm 88
record2 Kitty Final 92
record3 Bob Midterm 75
record4 Bob Final 81
record5 Jerry Midterm 90 
# safe modification using copy()

col = frame3["score"].copy()

col =  [93, 95, 70, 80, 88]

frame3["updated_score"] = col # add a new column

frame3 # original "score" column remains unchanged
student exam score updated_score
record1 Kitty Midterm 88 93
record2 Kitty Final 92 95
record3 Bob Midterm 75 70
record4 Bob Final 81 80
record5 Jerry Midterm 90 88

Deleting a column

The del keyword will delete columns as it does with a dictionary.

del frame3["score"] 

frame3
student exam updated_score
record1 Kitty Midterm 93
record2 Kitty Final 95
record3 Bob Midterm 70
record4 Bob Final 80
record5 Jerry Midterm 88

Transposing a DataFrame

When creating a DataFrame from a dictionary, the DataFrame can be transposed. The keys in the inner dicts are unioned and sorted to form the index in the result.

frame4 = pd.DataFrame(data)

frame4.T
0 1 2 3 4
student Kitty Kitty Bob Bob Jerry
exam Midterm Final Midterm Final Midterm
score 88 92 75 81 90 
frame3.T
record1 record2 record3 record4 record5
student Kitty Kitty Bob Bob Jerry
exam Midterm Final Midterm Final Midterm
updated_score 93 95 70 80 88
