Working with Lists

Looping allows you to repeat the same action—or a sequence of actions—for every item in a list. This becomes especially powerful when you’re working with long datasets. Instead of writing the same line of code over and over again, you write a loop once, and Python handles each item automatically.

Loop Through an Entire List

The most common way to process all items in a list is with a for loop. A for loop steps through the list one item at a time and lets you run code on each item.

Using our previous schools list as an exmple:

schools = ['Beaver Elementary School', 'Tree House Elementary School', 'Moonlight Middle School']

for school in schools:   
    print(school)

Avoiding Indentation Errors

Python uses indentation to determine how lines of code relate to one another. When you write code that contains loops or conditionals, indentation becomes essential for telling Python which lines belong together. As a result, beginners commonly run into two problems:

  • Indenting lines that shouldn’t be indented Forgetting to indent lines that should be indented

  • Seeing these examples now will help you avoid these mistakes and recognize them quickly in your own programs. Let’s walk through the most common indentation errors.

Using the range() Function

Python’s range() function makes it easy to generate a series of numbers. For example, you can use the range() function to print a series of numbers like this:

for number in range(1, 5):
    print(number)

for number in range(1, 6):

We can create almost any set of numbers you want to using the range() function. Consider how you might make a list of the first 10 square numbers (that is, the square of each integer from 1 through 10).

# first, create an empty list

for n in range(1, 11):

  squares.append(square)

List Comprehensions

The earlier approach for generating the list squares required several lines of code: you created an empty list, calculated each value in a loop, and appended the result manually. A list comprehension allows you to generate the same list in just one line of code. A list comprehension combines:

  • the expression that produces each value,

  • the for loop, and

  • the automatic appending of each new element

all in one compact, readable structure.

The following example builds the same list of square numbers you saw earlier, but using a list comprehension:

squares = [n**2 for n in range(1, 11)]

print(squares)
[1, 4, 9, 16, 25, 36, 49, 64, 81, 100]

Working with Part of a List

Slicing a List

In addition to working through all the elements in a list, you can also work with a specific group of items.

In Python, this group is called a slice. A slice lets you access a subset of a list, such as the first three items, the middle portion, or everything except the first element.

For example:

schools = ['Beaver Elementary School', 'Tree House Elementary School',
'Moonlight Middle School', 'Blue Ridge High School', 'Valley Creek High School']

# print the first three schools
print(schools[0:3])
['Beaver Elementary School', 'Tree House Elementary School', 'Moonlight Middle School']
# print the second to the fifth
print(schools[1:4])
['Tree House Elementary School', 'Moonlight Middle School', 'Blue Ridge High School']
# print up to the fourth element
print(schools[:4]) # ommit the first index in a slice
['Beaver Elementary School', 'Tree House Elementary School', 'Moonlight Middle School', 'Blue Ridge High School']
# print from the third element and on
print(schools[2:]) # ommit the second index in a slice
['Moonlight Middle School', 'Blue Ridge High School', 'Valley Creek High School']
# print the last three elements
print(schools[-3:]) # using negative index
['Moonlight Middle School', 'Blue Ridge High School', 'Valley Creek High School']

Note:

Slice notation uses the same indexing rules you already learned but lets you specify a range of positions instead of a single position.

  • The first index is inclusive

  • The second index is exclusive

  • Either index can be omitted to mean “start at the beginning” or “go to the end”

Looping Through a Slice

We can use a slice in a for loop if you want to loop through a subset of the elements in a list.

schools = ['Beaver Elementary School', 'Tree House Elementary School',
           'Moonlight Middle School', 'Blue Ridge High School',
           'Valley Creek High School']

print("Here are the first 3 schools in the list;")
Here are the first 3 schools in the list;
# loop through the first three schools
for school in schools[:3]:
    print(school)
Beaver Elementary School
Tree House Elementary School
Moonlight Middle School
print("\nHere are the last 3 schools in the list:")

Here are the last 3 schools in the list:
# loop through the last two schools
for school in schools[-3:]:
    print(school)
Moonlight Middle School
Blue Ridge High School
Valley Creek High School

Copying a List

To copy a list, you can make a slice that includes the entire original list by omitting the first index and the second index ([:]). This tells Python to make a slice that starts at the first item and ends with the last item, producing a copy of the entire list. Here’s an example:

schools = ['Beaver Elementary School', 'Tree House Elementary School',
           'Moonlight Middle School', 'Blue Ridge High School',
           'Valley Creek High School']

# make a copy of the entire list
schools_copy = schools[:]

print(schools_copy)
['Beaver Elementary School', 'Tree House Elementary School', 'Moonlight Middle School', 'Blue Ridge High School', 'Valley Creek High School']

If we had simply set a new list equal to the current list, we would not produce two separate lists. Instead, both variables would point to the same list in memory. Here’s what that looks like:

my_friends = ['Alice', 'Ben', 'Carla']

# this does NOT make a copy
their_friends = my_friends

my_friends.append('Derek')
their_friends.append('Eva')

print(my_friends)
['Alice', 'Ben', 'Carla', 'Derek', 'Eva']
print(their_friends)
['Alice', 'Ben', 'Carla', 'Derek', 'Eva']
# see how both new names are added to the same list in memory.

Working with Tuples

Looping Through All Values in a Tuple

We can loop over all the values in a tuple using a for loop, just as we do with a list. The main difference is simply that the data cannot be changed once stored inside the tuple.

scores = (100, 97, 94, 88, 86, 79)

for value in scores:
    print(value)
100
97
94
88
86
79

Writing Over a Tuple

Although you can’t modify the elements of a tuple, you can assign a new tuple to the same variable. This allows you to “change” the collection by replacing the entire tuple. For example,

scores = (100, 97, 94, 88, 86, 79)

print("The outdated scores are:")

    print(value)
