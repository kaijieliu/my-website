Variables

A variable stores information in your program. Using a variable may add a tiny bit of work for Python behind the scenes, but it saves you a lot of effort - especially when you need to reuse the same information multiple times. Consider the following example. Suppose you want to print several sentences that all include the same list of names:

print("Hello, Tommy, Jerry, Mickey, Minnie, Kitty!")
Hello, Tommy, Jerry, Mickey, Minnie, Kitty!
print("How are you doing, Tommy, Jerry, Mickey, Minnie, Kitty?")
How are you doing, Tommy, Jerry, Mickey, Minnie, Kitty?
print("Is everyone here today? Tommy, Jerry, Mickey, Minnie, Kitty.")
Is everyone here today? Tommy, Jerry, Mickey, Minnie, Kitty.

Typing all those names over and over again is tedious and error‑prone. Instead, we can store the names in a variable and use that variable inside each print() statement.

It’s also a good idea to begin by printing the variable by itself — print(names) — just to check that you stored the value correctly before using it in more complex lines.

names = "Tommy, Jerry, Mickey, Minnie, Kitty"

print(names)
Tommy, Jerry, Mickey, Minnie, Kitty
print("Hello," + names + "!")
Hello,Tommy, Jerry, Mickey, Minnie, Kitty!
print("How are you doing, " + names + "?")
How are you doing, Tommy, Jerry, Mickey, Minnie, Kitty?
print("Is everyone here today? " + names + ".")
Is everyone here today? Tommy, Jerry, Mickey, Minnie, Kitty.

Now, if you ever want to change the names, you only need to update the variable once, and every line that uses it will automatically reflect the change.

Data Type

Data is simply information. When being processed or analyzed in a program, data is usually stored in a variable to be worked with. Python supports several kinds of data. Each data type behaves differently. The common types include:

  • Strings – text data

  • Integers – whole numbers

  • Floats – decimal numbers

  • Booleans – values of either True or False

Strings

A string is a sequence of characters—letters, digits, spaces, punctuation, or symbols—enclosed in quotation marks.

Here are some examples:

"Python is fun!"
'Python is fun!'
"12345"
'12345'
"Tommy, Jerry, Mickey"
'Tommy, Jerry, Mickey'

Even though "12345" contains digits, it is still a string, because it is enclosed in quotes.

Strings can be created using either single quotes ' ' or double quotes " ". Both are valid.

'This is a string.'
'This is a string.'
"This is also a string."
'This is also a string.'

Python treats them the same, although double quotes are often easier when the string contains an apostrophe.

"This is Mickey's packet."
"This is Mickey's packet."

Printing Strings

You can use the print() function to display strings.

print("This is Mickey's packet.")
This is Mickey's packet.

Strings can be assigned to variables and printed out.

morning = "Good morning,"
afternoon = "Good afternoon,"
evening = "Good evening,"
greet = "How are you?"
person = "Mickey!"

print(morning, person, greet)
Good morning, Mickey! How are you?
print(afternoon, person, greet)
Good afternoon, Mickey! How are you?
print(evening, person, greet)
Good evening, Mickey! How are you?

Strings with Methods

A method is an action that Python can perform on a piece of data. When a method is used with a string, it returns a modified version of that string, like changing the cases or removing white spaces, and so on.

Change the Cases

In the example below, we assign the string “Good Morning” to the variable message. The dot (.) after the variable name in expressions like message.title() tells Python to apply a method to the value stored in message. Here’s what each method does:

  • message.title() → Converts the string to title case (first letter of each word capitalized)

  • message.upper() → Converts all letters to uppercase

  • message.lower() → Converts all letters to lowercase

message = "Good Morning"

print(message.title())
Good Morning
print(message.upper())
GOOD MORNING
print(message.lower())
good morning

Strip Whitespace

Python provides several methods to remove unwanted whitespace from the beginning and/or end of a string. These are especially useful when handling user input, file data, or text copied from external sources. Different methods for stripping the whitespace include:

  • rstrip() → Ensures no whitespace exists at the right side of a string
favorite = "python   "
print(favorite)
python   
print(favorite.rstrip())
python
  • lstrip() → Ensures no whitespace exists at the left side of a string
favorite = "   python"
print(favorite)
   python
print(favorite.lstrip())
python
  • strip() → Ensures no whitespace exists at both sides of a string
print(favorite)
   python
print(favorite.strip())
python

Remove prefix

Sometimes you want to remove a known starting part of a string. Python’s removeprefix() method makes this simple.

filename = "2026_report_final.pdf"
print(filename)
2026_report_final.pdf
clean_name = filename.removeprefix("2026_")
print(clean_name)
report_final.pdf
url = "https://example.com/home"
print(url)
https://example.com/home
clean_url = url.removeprefix("https://")
print(clean_url)
example.com/home

f-strings

Very often, you’ll want to combine text with variables inside a single string. Python makes this easy using f‑strings, which stands for formatted strings. They let you insert variables directly into a string by placing them inside curly braces {}. To use an f‑string, place the letter f before the opening quotation mark. Using the above example, you’ll see that we no longer need the punctuation within the string variables and how we can arrange the variables to get the printout messages we want.

morning = "Good morning"
afternoon = "Good afternoon"
evening = "Good evening"
greet = "How are you"
person = "Mickey"

print(f"{morning}, {person}! {greet}?")
Good morning, Mickey! How are you?
print(f"{afternoon}, {person}! {greet}?")
Good afternoon, Mickey! How are you?
print(f"{evening}, {person}! {greet}?")
Good evening, Mickey! How are you?
print(f"{evening}! {greet}, {person}?")
Good evening! How are you, Mickey?

Numbers

Numbers appear everywhere in programming: keeping score in games, counting items, calculating statistics, working with dates, and representing data in graphs or tables. Python treats numbers differently depending on their type and how they are used. The two most common numeric types you’ll see are:

  • Integers (whole numbers)

  • Floats (decimal numbers)

Integers

Integers are whole numbers—positive, negative, or zero. They do not contain decimals. Examples:

10
10
-3
-3
0
0
42
42

You can store integers in variables just like strings:

score = 10
age = 21

print(score)
10
print(age)
21

Basic Math with Integers

print(5 + 3)   # addition
8
print(10 - 4)  # subtraction
6
print(6 * 7)   # multiplication
42
print(20 / 5)  # division (results in a float)
4.0
print(7 // 2)  # floor division (drops the decimal)
3
print(7 % 2)   # remainder (modulo)
1
print(2 ** 3)  # exponent (2 to the 3rd power)
8
Integer Division Note

  • A single slash / represents divisions and always produces a float

    4/2
    2.0

  • A double slash //: performs integer (floor) division

    9.45//2
    4.0

Floats

Floats (short for floating‑point numbers) represent numbers with decimal points. Examples:

3.14
3.14
0.0
0.0
-2.5
-2.5
10.75
10.75

You can perform math with floats the same way as with integers:

temperature = 98.6
print(temperature)
98.6
print(temperature + 1.4)
100.0
print(5.0 * 2.0)
10.0

Common Floating-Point Precision

Floats can’t always represent decimal numbers exactly. For example, you might expect 0.3 below. However, Python shows:

print(0.1 + 0.2)
0.30000000000000004

Mixing Integers and Floats

If you combine an integer and a float in an operation, Python automatically converts the result to a float.

3 + 2
5
3 + 2.0
5.0
2*3
6
2*3.0
6.0
3**2
9
3.0**2
9.0

Booleans

A Boolean is a special data type in Python that represents one of two values:

  • True

  • False

Booleans often come from comparisons, conditions, and logical expressions. They are essential for decision‑making in programs—especially when you start using if statements, loops, and filtering data.

Basic Boolean Values

You can assign Boolean values directly to variables:

is_python_fun = True
is_raining = False

print(is_python_fun)
True
print(is_raining)
False

Comparison Operators

Most Booleans are created by comparing values. Python returns True or False based on the comparison.

print(5 > 3)     # Greater than
True
print(2 < 1)     # Less than
False
print(10 == 10)  # Equal to
True
print(7 != 2)    # Not equal to
True

Comparing Strings

You can compare strings as well:

name = "minnie"

print(name == "MiNNie")
False
print(name == "Minnie")
False
print(name == "minnie")  # Case matters!
True

Boolean with Logical Operators

You can combine Booleans using logical operators:

  • and → both conditions must be true

  • or → at least one condition must be true

  • not → reverses a Boolean value

Examples:

age = 20

print(age > 18 and age < 30)   # True
True
print(age < 18 or age > 65)    # False
False
print(not (age > 18))          # False
False

Using Booleans in a Simple Decision

temperature = 75

is_warm = temperature > 70
print(is_warm)
True
if is_warm:
    print("It's a warm day!")
It's a warm day!

Type Conversion

Different kinds of data - such as strings, integers, floats, and Booleans - cannot always be used together directly. Sometimes you need to convert a value from one type to another. This process is called type conversion.

Built‑in Functions

Python provides simple built‑in functions for converting between common types:

  • int() → converts to an integer

  • float() → converts to a float

  • str() → converts to a string

  • bool() → converts to a Boolean

Converting Strings to Numbers

age_str = "20"

# convert to an integer
age_int = int(age_str)
print(age_int)
20
# convert to float
age_float = float(age_str)
print(age_float)
20.0

Converting Numbers to Strings

Errors arise when trying to mix incompatible types. For example, the code below will spit out error message as TypeError: can only concatenate str (not "int") to str.

score = 98

print("Your score is " + score)
TypeError: can only concatenate str (not "int") to str

To fix it, we need to convert the score into a string.

score_str = str(score)

print("Your score is " + score_str)
Your score is 98

Remark: the mixed types won’t be a problem if using f-strings.

print(f"Your score is {score}.")
Your score is 98.

Converting between Floats and Integers

x = 3.14159
y = int(x)
z = float(y)

print(f"x = {x}")
x = 3.14159
print(f"y = {y}")
y = 3
print(f"z = {z}")
z = 3.0

Converting Values to Booleans

The bool() function converts values to True or False based on standard rules:

print(bool(1))      # True
True
print(bool(0))      # False
False
print(bool("Hi"))   # True (non-empty string)
True
print(bool(""))     # False (empty string)
False
